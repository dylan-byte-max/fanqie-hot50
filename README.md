# 🍅 番茄小说热门榜单 Top50

每日自动追踪番茄小说书库「最热」排行榜前50名，展示排名变化趋势。

## 功能

- 📊 每日北京时间 15:00 自动更新
- 🔍 支持按书名/作者搜索
- 🏷 支持按男频/女频、连载/完结筛选
- 📈 展示排名变化（上升/下降/新进入）
- 📅 支持查看历史数据（保留90天）
- 📱 自适应移动端

## 数据字段

| 字段 | 说明 |
|------|------|
| 排名 | 当日最热榜排名 |
| 书名 | 小说名称 |
| 作者 | 作者名 |
| 频道 | 男频/女频 |
| 标签 | 分类标签 |
| 简介 | 内容概要 |
| 状态 | 连载中/完结 |
| 排名变化 | 相比前一天的变化 |

## 技术栈

- **爬虫**: Node.js（纯原生，零依赖）
- **前端**: 纯 HTML/CSS/JS（零框架）
- **自动化**: GitHub Actions（每日定时）
- **部署**: GitHub Pages（免费）
